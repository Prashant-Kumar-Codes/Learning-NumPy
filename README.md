# Learning-NumPy

This repository documents my journey of learning **NumPy**, one of the most fundamental libraries for numerical computing in Python. It contains structured code examples, practice snippets, and small projects to build a strong foundation for data science, machine learning, and AI.

---

## Purpose

The goal of this repository is to:

* Understand core NumPy concepts from basics to intermediate level
* Practice array operations and vectorized computations
* Build intuition for multi-dimensional data handling
* Prepare a strong base for Machine Learning and Deep Learning
